### 🚀 Prompt 1: Engenharia de Software e Modelagem Matemática

Atue como um engenheiro de software sênior Python especializado em modelagem matemática. O seu objetivo é criar uma calculadora epidêmica se baseada no modelo SEIR para modelar a difusão de doenças infectocontagiosas. 

ESPECIFICAÇÕES DO CÓDIGO: 

1) Linguagem: Python 3;

2) Modelo Matemático: Use o modelo epidemiológico SIR (Suscetíveis, Infectados, Recuperados), permita que tenha ajustes de taxa de transmissão (beta), taxa de recuperação (gama), população total, número inicial de infectados;

3) Tecnologia: Use NumPy (para cálculos númericos), Pandas (para organizar os dados), Matplotlib (para visualização), Stramlit (para criar uma dashboard moderna), Plotly (para gráficos). 

ENTREGA: Código completo em Python, instruções para rodar localmente, código em português (com comentários). No final, escreva uma breve explicação do funcionamento."

### 🎓 Prompt 2: Escrita de Artigo Científico (Paper)

"Com o código pronto, atue como um pesquisador acadêmico e escreva um pequeno artigo científico sobre a calculadora epidêmica no formato de Paper (em português). 

Divida esse artigo em: 

1) Título: Claro, conciso e profissional.

2) Resumo: Síntese de objetivos do software e seus resultados.

3) Introdução: Apresentar o propósito de modelar a difusão dessas doenças.

4) Metodologia: Detalhar as equações que foram utilizadas no código.

5) Resultados: Apresentar como uma interface dinâmica ajuda na visualização.

6) Conclusão: Concluir sobre como um software de simulação pode contribuir para a saúde pública.
